## 其他功能
上文展示了项目的基础使用,本项目还有许多其他的功能期待您的探索.
* 图像操作支持撤销,当操作生效后也可以通过"取消"按钮恢复原始图像.
* 支持选择标注框的边框尺寸与颜色
* 可以通过添加放大镜来局部放大某个部分,放大比例可调
* 支持图像按列优先与按行优先顺序切换
* 支持统一调整所有图像的尺寸,并导出所有图像.
* 批量重命名: 双击列表可以调整文件名,导出图像时会以列表中的名字为图像命名.
* 图像标注窗口可分离,对于大尺寸图形可以全屏进行标注.
